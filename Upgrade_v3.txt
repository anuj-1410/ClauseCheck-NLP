## PART 1 — BACKEND UPGRADES

---

### 1.1 Replace `en_core_web_trf` → `en_core_web_sm`

**Files Affected:** `entity_extractor.py`, `obligation_detector.py`, `responsibility_detector.py`

**Current State:**
- Using `en_core_web_trf` (spaCy Transformer Model with RoBERTa)
- Used for: tokenization, POS tagging, NER, dependency parsing

**Problem:**
- You are NOT using transformer embeddings or deep contextual NER features
- You are only using: dependency labels (`nsubj`, `dobj`), basic entity labels, token-level parsing
- You already use `sentence-transformers` for semantic understanding
- Running both = duplicate transformer stacks in memory (~unnecessary overhead)

**Action:**
- Replace `en_core_web_trf` with `en_core_web_sm` across all three files
- `en_core_web_sm` is fully sufficient for your actual usage (dependency labels + basic NER)

**Result:** Faster load times, lower RAM usage, no loss of functionality

---

### 1.2 Replace `paraphrase-multilingual-mpnet-base-v2` → `paraphrase-multilingual-MiniLM-L12-v2`

**Files Affected:** `risk_detector.py`, `contract_comparator.py`

**Current State:**
- Using MPNet model (~400–500MB) for semantic similarity detection
- Used for: paraphrased risk detection, clause comparison, cross-lingual matching (Hindi ↔ English)

**Problem:**
- MPNet is unnecessarily heavy for your use case
- High RAM consumption, slower inference

**Why Semantic Embedding Is Still Needed:**
```
Clause:   "The consultant shall bear unlimited financial responsibility."
Prototype: "Unlimited liability clause."
Regex:     ❌ Fails
Embedding: ✅ Succeeds
```

**Action:**
- Replace with `paraphrase-multilingual-MiniLM-L12-v2`
- Same purpose, same multilingual support, much lighter, faster

**Result:** Same semantic capability, significantly reduced model size and inference time

### 1.1 Replace `en_core_web_trf` with `en_core_web_sm`

**Files to change:** `entity_extractor.py`, `obligation_detector.py`, `responsibility_detector.py`

**Current Reality:**
All three files already have a try/except fallback — they try `en_core_web_trf` first and fall back to `en_core_web_sm`. The fallback logic works. The problem is that `en_core_web_trf` is attempted every cold start, slowing down initialisation and consuming GPU memory if available.

**What to change (same pattern in all three files, in the `_get_spacy()` function):**
```python
# CURRENT (in all three files):
try:
    _spacy_nlp = spacy.load("en_core_web_trf")
    logger.info("spaCy en_core_web_trf loaded.")
except OSError:
    _spacy_nlp = spacy.load("en_core_web_sm")
    logger.info("Using en_core_web_sm fallback.")

# REPLACE WITH:
_spacy_nlp = spacy.load("en_core_web_sm")
logger.info("spaCy en_core_web_sm loaded.")
```

Remove the entire try/except for `en_core_web_trf`. The fallback is no longer needed once `en_core_web_sm` is the primary.

**Result:** Faster startup, lower RAM, no functional regression.

### 1.3 Fix Clause Segmenter Overlap Check

**File:** `clause_segmenter.py`

**Current Reality:**
```python
# In _segment_by_sections(), around line 65:
if b["start"] - filtered[-1]["start"] > 20:
    filtered.append(b)
```
The 20-character gap check is fragile — a real boundary could legitimately be within 20 chars of another.

**Replace the filter block with a range intersection check:**
```python
# REPLACE THE ENTIRE FILTER BLOCK WITH:
filtered = [boundaries[0]]
for b in boundaries[1:]:
    prev = filtered[-1]
    # Only add if this boundary starts after the previous boundary's match ends
    if b["start"] >= prev["match_end"]:
        filtered.append(b)
    elif b["start"] > prev["start"]:
        # Partial overlap — keep the one with the longer match span
        if b["match_end"] > prev["match_end"]:
            filtered[-1] = b
boundaries = filtered
```

## PART 2 — NEW FILE: `analysis_aggregator.py`

---

### 2.1 Why This File Is Needed and What It Does

**Current Reality of `analyze.py`:**
The `/api/analyze` route already runs all 14 analysis services and stores a comprehensive result in Supabase. The stored `clause_analysis` object contains:
- `clauses` — from `clause_segmenter.py`: `[{id, text, section_number}]`
- `entities` — from `entity_extractor.py`: `{parties, dates, monetary_values, durations, legal_references}`
- `obligations` — from `obligation_detector.py`: `[{clause_id, text, strength, modal, party, action, deadline, condition, obligation_type, is_passive}]`
- `risks` — from `risk_detector.py`: `[{clause_id, risk_type, severity, description, matched_text, clause_text, risk_score, confidence, legal_note, detection_method, jurisdiction_note}]`
- `compliance` — from `compliance_checker.py`: `{compliance_score, found_clauses, missing_clauses, details[{clause_type, found, quality_score, quality_checks}], total_checked, total_found, total_missing}`
- `explanations` — from `explanation_generator.py`: `{risk_explanations, compliance_explanations, overall_summary}`
- `responsibility` — from `responsibility_detector.py`: `{passive_voice, vague_terms, missing_subjects, ambiguity_score, total_issues}`
- `timeline` — from `timeline_extractor.py`: `{events[{id, value, type, clause_id, section, description, context, category, parsed_date}], categories, total_events}`
- `plain_english` — from `llm_service.py`: `[{clause_id, original, simplified}]` (first 15 clauses only)

**The gap:** The stored raw data contains everything, but the frontend tabs need **derived and enriched** data that no existing service computes:
- Contract health score (derived from risk + compliance + imbalance combined)
- Party burden percentages (aggregated from individual obligation party assignments)
- Contract lifecycle dates (computed from timeline events using date arithmetic)
- Party role inference — Client vs Consultant (from obligation text patterns)
- Financial summary (aggregated from entity monetary_values + durations)
- Risk `impact_category`, `suggested_fix`, `negotiation_advice` (lookup-table enrichment)
- Compliance `consequence` and `suggested_language` for missing/weak clauses (lookup-table enrichment)
- Plain English `complexity_reduction_percentage`, `passive_voice_removed` (computed metrics)
- Ambiguity `why_vague`, `suggested_rewrite` (lookup-table enrichment)

def _compute_contract_health(risks: List, compliance: Dict, obligations: List) -> Dict:
    """
    Derive contract health score (0-100).
    Lower score = higher risk to the user.
    """
    score = 100

    high_count = sum(1 for r in risks if r.get("severity") == "high")
    medium_count = sum(1 for r in risks if r.get("severity") == "medium")
    score -= high_count * 12
    score -= medium_count * 4

    missing_count = compliance.get("total_missing", 0)
    score -= missing_count * 7

    burden = _compute_burden_analysis(obligations)
    if burden["imbalance_level"] == "High":
        score -= 8
    elif burden["imbalance_level"] == "Medium":
        score -= 4

    score = max(0, min(100, score))

    if score >= 75:
        label = "Low Risk"
    elif score >= 50:
        label = "Moderate Risk"
    else:
        label = "High Risk"

    return {
        "score": score,
        "label": label,
        "breakdown": {
            "risk_penalty": high_count * 12 + medium_count * 4,
            "compliance_penalty": missing_count * 7,
            "imbalance_penalty": 8 if burden["imbalance_level"] == "High" else (4 if burden["imbalance_level"] == "Medium" else 0),
        }
    }


def _compute_responsibility_split(obligations: List) -> Dict:
    """
    Derive per-party obligation percentages from the stored obligations list.
    obligation_detector assigns 'party' to each obligation — we aggregate here.
    """
    party_counts: Dict[str, int] = {}
    party_display: Dict[str, str] = {}
    total = 0

    for ob in obligations:
        party = ob.get("party", "").strip()
        if party and len(party) > 1:
            key = party.lower()
            party_counts[key] = party_counts.get(key, 0) + 1
            if key not in party_display:
                party_display[key] = party
            total += 1

    if total == 0:
        return {"parties": [], "imbalance_level": "Unknown"}

    sorted_parties = sorted(party_counts.items(), key=lambda x: x[1], reverse=True)
    result_parties = []
    for k, count in sorted_parties:
        result_parties.append({
            "name": party_display.get(k, k.title()),
            "role": "",
            "count": count,
            "percentage": round((count / total) * 100, 1),
        })

    if len(result_parties) >= 2:
        top_pct = result_parties[0]["percentage"]
        imbalance = "High" if top_pct >= 70 else ("Medium" if top_pct >= 55 else "Low")
    else:
        imbalance = "Unknown"

    return {"parties": result_parties, "imbalance_level": imbalance}


def _compute_burden_analysis(obligations: List) -> Dict:
    """
    Full burden analysis: counts, ratio, imbalance, unilateral flags.
    Reads from stored obligations (obligation_detector output).
    obligation_detector._detect_power_imbalance already classifies obligation_type
    per individual obligation. This function aggregates to contract level.
    """
    split = _compute_responsibility_split(obligations)
    parties = split["parties"]
    total = sum(p["count"] for p in parties)

    mandatory_count = sum(1 for o in obligations if o.get("strength") == "mandatory")
    mutual_count = sum(1 for o in obligations if o.get("obligation_type") == "mutual")
    unilateral_count = sum(1 for o in obligations if o.get("obligation_type") == "unilateral")

    burden_ratio = 0.0
    if len(parties) >= 2 and parties[1]["count"] > 0:
        burden_ratio = round(parties[0]["count"] / parties[1]["count"], 2)

    all_text = " ".join(o.get("text", "") for o in obligations).lower()

    unilateral_indemnity = bool(
        re.search(r"indemnif\w+.{0,50}(?:all|any|every)", all_text)
        and len(parties) >= 1 and parties[0].get("percentage", 0) > 60
    )
    unilateral_termination = bool(
        re.search(r"(?:may|can)\s+terminat.{0,50}(?:any\s+time|without\s+(?:cause|notice))", all_text)
    )
    unilateral_liability = bool(
        re.search(r"(?:shall\s+be\s+liable|bears?\s+(?:all|full)\s+(?:liability|risk))", all_text)
        and len(parties) >= 1 and parties[0].get("percentage", 0) > 65
    )

    dominant_name = parties[0]["name"].lower() if parties else ""
    top_critical = [
        o for o in obligations
        if o.get("strength") == "mandatory" and dominant_name in o.get("party", "").lower()
    ][:5]

    return {
        "total_obligations": total,
        "by_party": parties,
        "burden_ratio": burden_ratio,
        "imbalance_level": split["imbalance_level"],
        "mandatory_count": mandatory_count,
        "mutual_count": mutual_count,
        "unilateral_count": unilateral_count,
        "unilateral_flags": {
            "unilateral_indemnity": unilateral_indemnity,
            "unilateral_termination": unilateral_termination,
            "unilateral_liability": unilateral_liability,
        },
        "top_critical_obligations": top_critical,
    }


def _infer_party_roles(entities: Dict, obligations: List) -> List[Dict]:
    """
    Infer party roles (Client / Consultant) from entity data and obligation patterns.
    entity_extractor returns entities["parties"] = [{text, label}].
    """
    raw_parties = entities.get("parties", [])
    if not raw_parties:
        return []

    payment_kw = {"pay", "payment", "remunerate", "compensate", "fee"}
    service_kw = {"deliver", "provide", "perform", "execute", "complete", "submit"}

    party_pay: Dict[str, int] = {}
    party_svc: Dict[str, int] = {}

    for ob in obligations:
        party = ob.get("party", "").strip().lower()
        text = ob.get("text", "").lower()
        if not party:
            continue
        if any(kw in text for kw in payment_kw):
            party_pay[party] = party_pay.get(party, 0) + 1
        if any(kw in text for kw in service_kw):
            party_svc[party] = party_svc.get(party, 0) + 1

    result = []
    for i, entity in enumerate(raw_parties[:5]):
        name = entity.get("text", "").strip()
        if not name or len(name) < 2:
            continue

        name_lower = name.lower()
        pay = sum(v for k, v in party_pay.items() if name_lower in k or k in name_lower)
        svc = sum(v for k, v in party_svc.items() if name_lower in k or k in name_lower)

        if pay > svc:
            role = "Client"
        elif svc > pay:
            role = "Service Provider / Consultant"
        else:
            role = "Client" if i == 0 else "Consultant"

        result.append({
            "name": name,
            "type": entity.get("label", ""),
            "inferred_role": role,
        })

    return result


def _compute_financial_summary(entities: Dict, timeline_events: List) -> Dict:
    """
    Aggregate financial data from entity_extractor output.
    monetary_values = [{text, label}], durations = [{text, label}]
    """
    monetary = entities.get("monetary_values", [])
    durations = entities.get("durations", [])

    amounts = []
    currency = "INR"

    for m in monetary:
        text = m.get("text", "")
        if any(s in text for s in ["₹", "INR", "Rs"]):
            currency = "INR"
        elif any(s in text for s in ["$", "USD"]):
            currency = "USD"

        nums = re.findall(r"[\d]+(?:\.\d{1,2})?", text.replace(",", ""))
        if nums:
            try:
                val = float(nums[0])
                if "lakh" in text.lower():
                    val *= 100000
                elif "crore" in text.lower():
                    val *= 10000000
                amounts.append(val)
            except ValueError:
                pass

    amounts.sort()
    monthly_amount = amounts[0] if amounts else None
    lump_sum = amounts[-1] if len(amounts) > 1 else None

    duration_months = None
    for d in durations:
        text = d.get("text", "").lower()
        m = re.search(r"(\d+)", text)
        if m:
            num = int(m.group(1))
            if "year" in text:
                duration_months = num * 12
                break
            elif "month" in text:
                duration_months = num
                break

    total_value = None
    if monthly_amount and duration_months:
        total_value = monthly_amount * duration_months

    return {
        "monthly_amount": monthly_amount,
        "lump_sum_amount": lump_sum,
        "currency": currency,
        "duration_months": duration_months,
        "total_estimated_value": total_value,
        "liability_cap_defined": False,  # updated by caller via _check_liability_cap
        "liability_cap_amount": None,
        "raw_monetary_values": [m.get("text") for m in monetary],
    }


def _check_liability_cap_from_compliance(compliance: Dict) -> bool:
    """
    Check compliance structural validators for liability cap.
    compliance_checker STRUCTURAL_VALIDATORS["liability"]["checks"] includes "cap_defined".
    """
    for detail in compliance.get("details", []):
        if detail.get("clause_type") == "liability":
            for qc in detail.get("quality_checks", []):
                if qc.get("name") == "cap_defined" and qc.get("passed"):
                    return True
    return False


def _compute_lifecycle(timeline_events: List) -> Dict:
    """
    Derive contract lifecycle from timeline events.
    timeline_extractor returns events with parsed_date (ISO YYYY-MM-DD) and type fields.
    """
    from datetime import datetime, timedelta

    start_date = None
    end_date = None
    notice_period_days = None
    duration_months = None

    for ev in timeline_events:
        etype = ev.get("type", "")
        parsed = ev.get("parsed_date")
        value = ev.get("value", "").lower()
        desc = ev.get("description", "").lower()

        if parsed and etype == "date" and start_date is None:
            if any(kw in desc for kw in ["start", "commence", "effective", "begin"]):
                start_date = parsed

        if etype == "duration":
            m = re.search(r"(\d+)\s*(?:month|year|week|day)", value, re.IGNORECASE)
            if m:
                num = int(m.group(1))
                if "year" in value:
                    duration_months = num * 12
                elif "month" in value:
                    duration_months = num

        if etype == "notice":
            m = re.search(r"(\d+)", value)
            if m:
                num = int(m.group(1))
                notice_period_days = num * 30 if "month" in value else (num * 7 if "week" in value else num)

    # Compute end date
    if start_date and duration_months:
        try:
            from dateutil.relativedelta import relativedelta
            start_dt = datetime.strptime(start_date, "%Y-%m-%d")
            end_date = (start_dt + relativedelta(months=int(duration_months))).strftime("%Y-%m-%d")
        except Exception:
            pass

    # Compute notice deadline
    notice_deadline = None
    if end_date and notice_period_days:
        try:
            end_dt = datetime.strptime(end_date, "%Y-%m-%d")
            notice_deadline = (end_dt - timedelta(days=notice_period_days)).strftime("%Y-%m-%d")
        except Exception:
            pass

    # Build key dates
    key_dates = []
    seen = set()

    def _add_date(event_name, date_val, urgency="normal"):
        k = (event_name, date_val)
        if date_val and k not in seen:
            seen.add(k)
            key_dates.append({"event": event_name, "date": date_val, "urgency": urgency})

    _add_date("Contract Start", start_date)
    _add_date("Contract End", end_date)
    _add_date("Notice Deadline", notice_deadline, "high")

    for ev in timeline_events:
        if ev.get("parsed_date") and ev.get("type") == "date":
            _add_date(ev.get("description", "Date Reference"), ev["parsed_date"])

    return {
        "contract_start": start_date,
        "contract_end": end_date,
        "notice_deadline": notice_deadline,
        "notice_period_days": notice_period_days,
        "duration_months": duration_months,
        "renewal_deadline": None,
        "post_termination_obligations": [],
        "critical_deadlines": [kd for kd in key_dates if kd["urgency"] == "high"],
        "key_dates_table": key_dates,
    }


def _enrich_risks(risks: List) -> List[Dict]:
    """
    Add impact_category, exposure_level, suggested_fix, negotiation_advice
    to each risk. These fields do not exist in risk_detector.py output.
    """
    enriched = []
    for r in risks:
        rt = r.get("risk_type", "")
        enriched.append({
            **r,
            "impact_category":    RISK_IMPACT_CATEGORY.get(rt, "Operational"),
            "exposure_level":     RISK_EXPOSURE_LEVEL.get(rt, "Undefined"),
            "suggested_fix":      RISK_SUGGESTED_FIX.get(rt, "Review and negotiate this clause with qualified legal counsel."),
            "negotiation_advice": RISK_NEGOTIATION_ADVICE.get(rt, "Seek balanced and mutual terms for both parties."),
        })
    return enriched


def _enrich_compliance(compliance: Dict) -> List[Dict]:
    """
    Enrich compliance details with balanced/enforceable flags, consequence, priority, suggested_language.
    compliance_checker returns quality_score per clause — we derive status flags from it.
    """
    enriched = []
    for detail in compliance.get("details", []):
        ct = detail.get("clause_type", "")
        found = detail.get("found", False)
        quality = detail.get("quality_score", 1.0)

        if not found:
            status = "missing"
            balanced = enforceable = structured_properly = None
            risk_level = "high" if detail.get("weight", 0) >= 9 else ("medium" if detail.get("weight", 0) >= 7 else "low")
        elif quality >= 0.8:
            status = "present"
            balanced = enforceable = structured_properly = True
            risk_level = "low"
        else:
            status = "weak"
            balanced = quality >= 0.6
            enforceable = quality >= 0.5
            structured_properly = False
            risk_level = "medium"

        enriched.append({
            **detail,
            "status": status,
            "balanced": balanced,
            "enforceable": enforceable,
            "structured_properly": structured_properly,
            "risk_level": risk_level,
            "consequence": COMPLIANCE_CONSEQUENCES.get(ct) if not found else None,
            "priority": "High" if risk_level == "high" else ("Medium" if risk_level == "medium" else None),
            "suggested_language": COMPLIANCE_SUGGESTED_LANGUAGE.get(ct) if (not found or status == "weak") else None,
        })

    order = {"missing": 0, "weak": 1, "present": 2}
    enriched.sort(key=lambda x: order.get(x["status"], 3))
    return enriched


def _enrich_plain_english(plain_english: List, clauses: List, risks: List, responsibility: Dict) -> List[Dict]:
    """
    Add complexity metrics to plain_english entries.
    plain_english comes from analyze.py: [{clause_id, original, simplified}]
    """
    risk_map: Dict[int, str] = {}
    for r in risks:
        cid = r.get("clause_id")
        sev = r.get("severity", "low")
        if cid and (cid not in risk_map or sev == "high"):
            risk_map[cid] = sev

    passive_clause_ids = {p.get("clause_id") for p in responsibility.get("passive_voice", [])}

    enriched = []
    for pe in plain_english:
        original = pe.get("original", "")
        simplified = pe.get("simplified", "")
        cid = pe.get("clause_id")

        orig_words = max(len(original.split()), 1)
        simp_words = len(simplified.split()) if simplified else 0
        reduction = max(0, round(((orig_words - simp_words) / orig_words) * 100))

        orig_sents = max(len(re.split(r'[.!?।]', original)), 1) if original else 0
        simp_sents = len(re.split(r'[.!?।]', simplified)) if simplified else 0

        enriched.append({
            **pe,
            "complexity_reduction_percentage": reduction,
            "passive_voice_removed": cid in passive_clause_ids,
            "original_sentence_count": orig_sents,
            "simplified_sentence_count": simp_sents,
            "risk_level": risk_map.get(cid, "low"),
        })
    return enriched


def _enrich_ambiguity(responsibility: Dict) -> Dict:
    """
    Enrich vague_terms and passive_voice with lookup-table fields.
    responsibility_detector already returns the base fields —
    this adds why_vague, suggested_rewrite, active_rewrite from tables above.
    """
    enriched_vague = []
    for vt in responsibility.get("vague_terms", []):
        term = vt.get("term", "").lower()
        enriched_vague.append({
            **vt,
            "why_vague":        VAGUE_TERM_WHY.get(term, vt.get("issue", "This term lacks a precise definition.")),
            "suggested_rewrite": VAGUE_TERM_REWRITE.get(term, vt.get("suggestion", "Provide a specific, measurable definition.")),
            "preview_text":     vt.get("preview_text", vt.get("context", "")),
        })

    enriched_passive = []
    for pv in responsibility.get("passive_voice", []):
        enriched_passive.append({
            **pv,
            "active_rewrite": pv.get("active_rewrite", f"[Specify responsible party] shall {pv.get('matched_text', 'perform the required action')}."),
        })

    return {**responsibility, "vague_terms": enriched_vague, "passive_voice": enriched_passive}

    medium = sum(1 for r in risks if r.get("severity") == "medium")

    vague_count   = len(responsibility.get("vague_terms", []))
    passive_count = len(responsibility.get("passive_voice", []))
    affected_ids  = set(
        v.get("clause_id") for v in responsibility.get("vague_terms", [])
    ) | set(p.get("clause_id") for p in responsibility.get("passive_voice", []))

    top_risks = sorted(risks, key=lambda r: {"high": 0, "medium": 1, "low": 2}.get(r.get("severity"), 3))[:3]
    priority_actions = []
    for r in top_risks:
        priority_actions.append(f"Address {r.get('risk_type', 'risk').replace('_', ' ')} in Clause {r.get('clause_id')}")
    if missing_names:
        priority_actions.append(f"Add missing clauses: {', '.join(missing_names[:2])}")

    dominant_party = resp_split["parties"][0] if resp_split and resp_split.get("parties") else None
    balance_text = (
        f"Skewed toward {dominant_party['name']} ({dominant_party['percentage']}%)"
        if dominant_party else "Unable to determine"
    )

    return {
        "contract_health": health,
        "responsibility_split": resp_split,
        "risk_distribution": {"high": high, "medium": medium, "low": low, "total": len(risks)},
        "compliance_summary": {
            "present_and_strong": present_strong,
            "present_but_weak": present_weak,
            "missing": missing_count,
            "total_checked": compliance.get("total_checked", 0),
        },
        "ambiguity_summary": {
            "total_vague_terms": vague_count,
            "total_passive_voice": passive_count,
            "affected_clauses": len(affected_ids),
        },
        "critical_gaps": {
            "missing_clause_count": missing_count,
            "high_severity_risk_count": high,
            "missing_clauses": missing_names,
        },
        "financial_exposure": financial,
        "executive_summary": {
            "contract_type": ca.get("contract_type", {}).get("name", "Unknown"),
            "parties": parties_with_roles,
            "overall_assessment": explanations.get("overall_summary", ""),
            "top_risks": [{"severity": r.get("severity"), "description": r.get("description")} for r in top_risks],
            "missing_clauses": missing_names,
            "obligation_balance": balance_text,
            "priority_actions": priority_actions,
        },
    }

    compliance = ca.get("compliance", {})
    clauses    = ca.get("clauses", [])
    enriched   = _enrich_compliance(compliance)

    # Map each compliance item to the actual clause text via matched_keyword
    for item in enriched:
        item["clause_id"] = None
        item["clause_text"] = None
        if item.get("found") and item.get("matched_keyword"):
            kw = item["matched_keyword"].lower()
            for clause in clauses:
                if kw in clause.get("text", "").lower():
                    item["clause_id"] = clause["id"]
                    item["clause_text"] = clause["text"][:300]
                    break

    weak_count = sum(1 for d in enriched if d.get("status") == "weak")

    return {
        "compliance_items": enriched,
        "summary": {
            "present_and_strong": compliance.get("total_found", 0) - weak_count,
            "present_but_weak": weak_count,
            "missing": compliance.get("total_missing", 0),
            "total_checked": compliance.get("total_checked", 0),
            "compliance_score": compliance.get("compliance_score", 0),
        },
    }

# ROUTE: Re-Analyze Single Clause (What-If)
# ═══════════════════════════════════════════════════════════════════

class ReanalyzeRequest(BaseModel):
    clause_id: int
    modified_text: str


@router.post("/{analysis_id}/reanalyze-clause")
async def reanalyze_clause(analysis_id: str, req: ReanalyzeRequest):
    """
    Re-run risk + ambiguity analysis on a single modified clause.
    Returns before/after comparison. Does NOT re-run the full pipeline.
    """
    result = _load_result(analysis_id)
    ca = _get_ca(result)

    clauses = ca.get("clauses", [])
    original = next((c for c in clauses if c["id"] == req.clause_id), None)
    if not original:
        raise HTTPException(status_code=404, detail=f"Clause {req.clause_id} not found.")

    orig_risks = [r for r in ca.get("risks", []) if r.get("clause_id") == req.clause_id]
    orig_risk_score = sum(r.get("risk_score", 0) for r in orig_risks)
    orig_ambiguities = [v for v in ca.get("responsibility", {}).get("vague_terms", []) if v.get("clause_id") == req.clause_id]
    orig_passives = [p for p in ca.get("responsibility", {}).get("passive_voice", []) if p.get("clause_id") == req.clause_id]

    try:
        from services.risk_detector import detect_risks
        from services.responsibility_detector import detect_responsibility_issues

        modified_list = [{"id": req.clause_id, "text": req.modified_text, "section_number": ""}]
        new_risks = detect_risks(modified_list, "en")
        new_risk_score = sum(r.get("risk_score", 0) for r in new_risks)

        new_resp = detect_responsibility_issues(modified_list, "en")
        new_ambiguities = new_resp.get("vague_terms", [])
        new_passives = new_resp.get("passive_voice", [])

    except Exception as e:
        logger.error(f"Re-analysis failed: {e}")
        raise HTTPException(status_code=500, detail=f"Re-analysis failed: {str(e)}")

    diff = orig_risk_score - new_risk_score
    improvement = (
        f"Risk score reduced by {diff} points." if diff > 0
        else (f"Risk score increased by {abs(diff)} points." if diff < 0
              else "No change in risk score.")
    )

    return {
        "clause_id": req.clause_id,
        "original_text": original.get("text", "")[:400],
        "modified_text": req.modified_text[:400],
        "before": {
            "risk_score": orig_risk_score,
            "risk_count": len(orig_risks),
            "risks": orig_risks,
            "ambiguity_count": len(orig_ambiguities),
            "passive_count": len(orig_passives),
        },
        "after": {
            "risk_score": new_risk_score,
            "risk_count": len(new_risks),
            "risks": new_risks,
            "ambiguity_count": len(new_ambiguities),
            "passive_count": len(new_passives),
        },
        "improvement_summary": improvement,
    }
